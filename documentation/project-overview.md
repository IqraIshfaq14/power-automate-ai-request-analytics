# Project Overview

This project demonstrates an AI-powered request automation system using Microsoft Power Automate and AI Builder.

The system is designed to process incoming business requests from email, extract key information, classify requests using AI, route them based on business rules, and store all results in SharePoint for tracking and analytics.

## Main Objectives

- Reduce manual request triage
- Improve request visibility
- Standardize inconsistent email requests
- Use AI to support classification and summarization
- Create structured data for reporting and analytics
- Support approvals, escalation, SLA tracking, and audit history

## Main Components

- Outlook email trigger
- Power Automate cloud flow
- HTML-to-text conversion
- Expressions for field extraction
- AI Builder prompt
- SharePoint tracking list
- Teams notifications
- Planner task creation
- Approval workflow
- SLA monitoring
- Duplicate detection
